E-Commerce Platform

A full-featured e-commerce site built with React, Node.js, and MongoDB.

Live Demo View on GitHub

About The Project

This project is a comprehensive e-commerce solution designed to provide a seamless shopping experience. It includes robust features for both customers and administrators. The primary goal was to create a high-performance, secure, and scalable platform using the MERN stack. I focused on creating a clean user interface and an intuitive administrative dashboard for managing products, orders, and users.

Key Features

  • Full-featured shopping cart with quantity management.
  • Secure user authentication and profile management.
  • Product reviews and rating system.
  • Admin panel for product and order management.
  • Integration with Stripe for secure payment processing.

Technologies Used

  • Front-End: React, Redux for state management, Axios for API calls.
  • Back-End: Node.js, Express.js for the REST API.
  • Database: MongoDB with Mongoose for data modeling.
  • Authentication: JSON Web Tokens (JWT).
  • Deployment: Hosted on a cloud platform with CI/CD pipeline.
